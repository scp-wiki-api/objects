**SCP-9831:** MYMEMORY

**Author:** 

*And now I’m listening to some ‘Foundation’ in the sky… Oh! You’re… down here now. Okay.*

A project I've been working on since June 2025. I wanted to make something weird, unique, and most importantly, fun!

Buy MYMEMORY on [*https://store.steampowered.com/app/4506990/MYMEMORY/ Steam] today!

System requirements:

**All platforms:** 4GB ram, 1920x1080 resolution.

**Desktop:** Intel i5-2xxx, AMD FX 4xxx, any AMD Ryzen. Dedicated GPU, WebGL support.

**Mobile is not recommended!**

I have tested the game 100% working on a 2011 iMac. If you're newer than that you're probably fine.

**25Mb/s or better** internet to avoid load times. The game does its best to preload assets, but it isn't perfect.

**Hardware acceleration should be enabled in your browser!**

If you get stuck or find a bug, please let me know in the comments.

If you need to clear your save file, start a new game while holding Shift+Space.

If the game is too hard, turn on "Easy Mode" in the options menu.

All credits in the license box.
