**Item #:** SCP-8072

**Object Class:** Safe

**Special Containment Procedures:** SCP-8072 may be kept in any standard storage locker when not being tested.

**Description:** SCP-8072 is a steel garbage can. SCP-8072 has several properties that a garbage can normally cannot have:

# SCP-8072 wears glasses.

# SCP-8072 is a picky eater.

# SCP-8072 is polyamorous.

# SCP-8072 travels for work.

# SCP-8072 is monogamous.The contradiction with property #3 is noted.

# SCP-8072 is very flirtatious.

# SCP-8072 has dyed red hair.

# SCP-8072 enjoys country music.

# SCP-8072 is attracted to women.

# SCP-8072 has been married twice.

# SCP-8072 uses recreational drugs.

# SCP-8072 is disgusted by anal sex.

# SCP-8072 is a high school dropout.

Note that SCP-8072 has *no* unusual properties besides those listed here, including those that would normally be prerequisites for the stated anomalies. For example, SCP-8072 has no hair, despite having dyed red hair. This suggests that these properties are extrinsic, rather than intrinsic, to SCP-8072.
